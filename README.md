# Vergil

`/ˈvɜːrdʒɪl/`


A editor written with [Inferno](https://www.infernojs.org/) + [ProseMirror](http://prosemirror.net/)

```txt
\  /_ .__ o|
 \/(/_|(_|||
        _|
```


## Build

```zsh
% npm run build
```


## Test

### Unit Test

```zsh
: with tape
% npm test
```

### E2E Test

```zsh
: via karma
% npm run karma
```
