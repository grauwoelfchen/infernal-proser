# Vergil

`/ˈvɜːrdʒɪl/`


A editor written with [Inferno](https://www.infernojs.org/) + [ProseMirror](http://prosemirror.net/)

```txt
\  /_ .__ o|
 \/(/_|(_|||
        _|
```

## Build

```zsh
% npm run build
```
